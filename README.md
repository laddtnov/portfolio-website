## 📱 Responsive Design

The website is fully responsive with breakpoints at:

| Device | Breakpoint | Layout |
|--------|-----------|--------|
| 🖥️ Desktop | > 768px | Full grid layout |
| 📱 Tablet | ≤ 768px | Stacked layout |
| 📱 Mobile | ≤ 480px | Single column |
